ソースコードを書く時のルール


1. 文字列以外は、半角英数字で書く
2. 大文字と小文字は区別される
3. 文の最後はセミコロンでおわる
 - 改行ではない
 
 Javaのコーディング規約
 
 1. クラス名の先頭は大文字にする
 
 2. メソッド名・変数名の先頭は小文字にする
 
 3. メソッド名は、動司　+　名刺
  
  4. 定数名は大文字
  
  FLAG_ON
